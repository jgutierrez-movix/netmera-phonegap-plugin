public static void init (Application application, NetmeraProperties properties)

Authenticates user and application. It is recommended to call this method at the beginning of the program, from Application class of your app.

This method get NetmeraProperties and initializes the application. If you entered a google project number and activity class name in properties instance and if necessary additions for push are added to AndroidManifest.xml file, your app will automatically be registered to push notification service of Netmera.

public static void init (Application application)

Authenticates user and application. It is recommended to call this method at the beginning of the program, from Application class of your app.

This method reads assets/netmera.properties file and initializes the application. If you entered a google project number and activity class name in netmera.properties file and if necessary additions for push are added to AndroidManifest.xml file, your app will automatically be registered to push notification service of Netmera.
